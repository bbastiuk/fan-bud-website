# Шрифти

Ця папка порожня — сайт зараз використовує Google Fonts (Oswald + Bebas Neue + Archivo).

## Якщо маєш ліцензійний BENZIN

Поклади сюди файли:
- `Benzin-Regular.woff2`
- `Benzin-Medium.woff2`
- `Benzin-SemiBold.woff2`
- `Benzin-Bold.woff2`

(або інші weights які маєш)

Потім додай `@font-face` правила у `css/styles.css` — інструкція у головному README.md.
